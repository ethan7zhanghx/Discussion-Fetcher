支持导入 Twitter CSV 文件（帖子和评论）

用于标记导入的数据，方便后续筛选
可以选择多个文件同时导入
CSV 格式要求

必填字段：

  • ID - 推文ID
  • 链接 - 推文URL
  • 发布日期 - 发布时间
  • 内容 - 推文内容
  • 用户名 - 发推用户

如何区分帖子和评论：

  • 帖子：回复推文 ID 为空
  • 评论：回复推文 ID 有值
